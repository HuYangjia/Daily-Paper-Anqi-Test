ILA-MCM: Integrating Memory Consistency Models with Instruction-Level Abstractions for Heterogeneous SoC Verification

基本信息：FMCAD 2018 · Zhang, Trippel 等 · 检索入口
一句话核心贡献：把 ILA 与 MCM 连接起来做异构 SoC 全局行为验证。
背景与问题：SoC 中多个 IP 各自正确，但组合后可能违反全局一致性语义。
方法详解：
  • 用 ILA 表达指令级行为抽象。
  • 用 MCM 约束并发可观测顺序。
  • 在统一框架中进行系统级一致性验证。
关键结果：提高了异构系统验证可组合性，为后续协议/缓存一致性工作提供方法基础。
零基础学习建议：先补 ILA 基础，再看论文如何把 ILA 与 axiomatic MCM 接口化。
